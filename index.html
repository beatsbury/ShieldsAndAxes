This project is my take on several fields of study simultaneously. Those are:

  • C# programming, and more specifically, WPF technology
  • Git as a mean to control my project
  • GitHub as a collaboration and distribution system

"Shields And Axes" is a game of "Tic-tac-toe" using the viking theme. As of now, main application is fully functional. It supports 1 player vs. CPU and 2 player modes. In planning for now is migration of the project to MVVM pattern (that should be interesting) and adding sounds (even more interesting).
